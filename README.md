## A simple webapp to pull data from the Eventbrite, Parkwhiz, GoogleMaps, Google Places APIs and display them on a map for a more streamlined meetup experience.

## Technologies Used: 
* HTML
* CSS
* Bootstrap
* Jquery
* JSON
* JS
* Adobe Photoshop.
